Textos

Existem diversas tags para marcarmos textos em HTML. Para títulos, existe a família de tags que vai de h1 (mais relevante) até h6 (menos relevante).

Para os parágrafos de conteúdo, existe a tag p e, ao longo do texto escrito dentro de p, é possível utilizar tags que, de alguma forma, diferenciam o conteúdo:

  • Strong e "b": negrito
  • "i": itálico
  • "u": sublinhado
  • "strike": riscado

Listas

As listas, em HTML, podem ser ordenadas ou não. As listas ordenadas são delimitadas pela tag "ol" enquanto as não ordenadas são delimitadas por "ul". Em ambas, os itens são delimitados por "li", veja:

Lista ordenada:

  1. Primeiro item
  2. Segundo item

Lista não ordenada:

  • Primeiro item
  • Segundo item

Com estas tags, é possível encadear listas:

  1. Primeiro item
    • Primeiro item secundário
    • Segundo item secundário

Links

É muito comum que utilizemos links em páginas web. Em HTML isso é feito com a tag "a". Alguns de seus atributos principais são:

  • href: indica o link a ser aberto.
  • target: indica onde o link sera aberto. Os principais valores que pode assumir são:
    • "_blank": abre em uma nova guia do navegador;
    • "_self": abre na guia em que o atual HTML está disposto.

Neste link, o Google será aberto em uma nova guia.
Neste link, o Google será aberto na guia atual.

Podemos também redirecionar o link para uma seção do próprio website que estamos fazendo.

Para isso, atribuímos um id para a seção que redirecionaremos e, no href da tag a passamos o valor "#" + id atribuído.
Exemplo: a seção "Listas" tem id "listas", portanto, atribuímos href="#listas" a este link.

Esta técnica é denominada ancoragem!
